命令名稱： 查找窗口句柄
命令功能： 查找窗口句柄
命令參數：
參數1 字符串型，窗口類名
參數2 字符串型，窗口標題
返回值： 整數型，窗口句柄
腳本例子：
按鍵精靈8及
以上語法 
MsgBox lib.API.查找窗口句柄("notepad",0)
命令名稱： 激活窗口並置前
命令功能： 激活窗口並置前
命令參數：
參數1 整數型，窗口句柄
返回值： 無
腳本例子：
按鍵精靈8及
以上語法 
窗口句柄 = lib.API.查找窗口句柄("notepad",0)
Call lib.API.激活窗口並置前(窗口句柄)
命令名稱： 設置窗口透明度
命令功能： 設置窗口透明度
命令參數：
參數1 整數型，窗口句柄
參數2 整數型，透明度（0~255）
返回值： 無
腳本例子：
按鍵精靈8及
以上語法 
//透明度：0~255
窗口句柄 = lib.API.查找窗口句柄("notepad",0)
Call lib.API.設置窗口透明度(窗口句柄,100)

命令名稱： 取消窗口設置
命令功能： 取消窗口設置
命令參數：
參數1 整數型，窗口句柄
參數2 整數型，模式（0，一般窗體；1，UpdateLayeredWindow畫窗體）
返回值： 無
腳本例子：
按鍵精靈8及
以上語法 
//模式：0，一般窗體；1，UpdateLayeredWindow畫窗體
窗口句柄 = lib.API.查找窗口句柄("notepad",0)
Call lib.API.取消窗口設置(窗口句柄,0)
